# 下载3D模型文件并集成到前端项目

## 任务目标
1. 下载一个可用的3D模型文件
2. 安装Three.js依赖
3. 创建3D模型展示组件
4. 在首页显示3D模型

## 具体步骤
1. **确定Three.js支持的模型格式**
   - Three.js支持多种格式：GLTF/GLB、OBJ、FBX、STL等
   - 推荐使用GLTF格式，它是Web上3D模型的标准格式

2. **下载3D模型文件**
   - 从免费3D模型资源网站下载一个GLTF格式的模型
   - 选择一个适合展示的模型（如角色、物体等）

3. **放置模型文件**
   - 将下载的模型文件放置到 `Frontend_main/public/models` 目录
   - 确保模型文件结构完整（包含必要的纹理和材质文件）

4. **安装Three.js依赖**
   - 在Frontend_main目录下运行 `npm install three`
   - 安装Three.js的加载器：`npm install @types/three`（可选）

5. **创建3D模型展示组件**
   - 在 `Frontend_main/src/components` 目录下创建 `ThreeModel.vue` 组件
   - 实现模型加载、场景设置、相机控制等功能

6. **集成到首页**
   - 修改 `Frontend_main/src/components/HelloWorld.vue` 文件
   - 引入并使用ThreeModel组件

7. **测试和优化**
   - 确保模型正确加载和显示
   - 优化模型渲染性能
   - 测试旋转和交互功能

## 预期结果
- 前端首页能够显示一个可旋转的3D模型
- 模型加载速度快，渲染效果良好
- 用户可以通过鼠标交互控制模型视角
