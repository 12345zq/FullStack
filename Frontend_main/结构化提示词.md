# 结构化提示词

## 任务背景
用户需要在Frontend_main目录中创建一个Vue 3项目，配置Docker环境，并确保debug.log和start文件被Git忽略。

## 任务步骤

### 1. 目录切换与项目初始化
- 切换到Frontend_main目录
- 使用Vite创建最新版本的Vue 3项目
- 初始化Git仓库

### 2. Docker配置
- 创建Dockerfile文件，使用Node.js 20-alpine作为构建环境，Nginx作为生产服务器
- 创建run-docker.ps1脚本，包含构建和运行Docker容器的命令
- 修复Dockerfile中的重复FROM指令
- 构建Docker镜像
- 运行Docker容器，映射80端口到主机的8080端口
- 验证容器状态

### 3. Git忽略配置
- 在Frontend_main目录中更新.gitignore文件，添加start文件
- 在父目录中创建.gitignore文件，添加debug.log和start文件
- 提交对.gitignore文件的更改，以及添加Dockerfile和run-docker.ps1文件到Git仓库

## 技术要求
- 使用最新版本的Vue 3
- 使用Node.js 20+版本（Vite要求）
- 使用Docker进行容器化部署
- 确保敏感文件和日志文件被Git忽略

## 预期结果
- Vue 3项目成功创建并在Docker容器中运行
- 容器映射到主机的8080端口
- debug.log和start文件被Git忽略
- 所有配置文件都已提交到Git仓库
